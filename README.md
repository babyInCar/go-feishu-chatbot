# 应用详情

本项目基于web框架 `gin`，构建飞书机器人并署到阿里云 Serverless 平台（函数计算 FC）。

> Gin 是速度极快的Go web框架，在编写高性能的web服务器或中间件具有无与伦比的优势。

通过 Serverless Devs 开发者工具，您只需要几步，就可以体验 Serverless 架构，带来的降本提效的技术红利。
